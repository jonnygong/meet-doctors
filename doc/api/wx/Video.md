# 微信端视频列表 接口文档

+ 作者：qqs

+ 相关表结构

+ 视频分类，视频列表

+ 调用： http://api.mtd.kfw001.ts/Wx/控制器/方法(247)

+ 参数前有*为必填

### 涉及的表结构

|  表名称  |  参数说明 |
| --------- |  ------- |
| mtd_category | 分类，包括专家分类，产品分类，视频分类 |
| mtd_video_list | 视频列表 |



### 微信端视频分类

+ __接口说明__： __用于获取视频分类__

+ __接口地址__： __/Wx/Video/category__

+ __请求参数__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |


+ __响应参数__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| id | int | 分类id |
| img | varchar | 图片 |
| name | varchar | 分类名称 |



### 微信端视频列表

+ __接口说明__： __用于获取微信端视频列表__

+ __接口地址__： __/Wx/Video/list__

+ __请求参数__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| *category_id | int | 分类id |

+ __响应参数__

|  参数名称  | 参数类型 | 参数说明 |
| --------- | -------- | ------- |
| id | int | 视频id |
| cover | varchar | 视频封面 |
| title | varchar | 视频标题 |
| long_time | varchar | 视频时长 |
